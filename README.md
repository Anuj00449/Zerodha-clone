# Zerodha-Clone

This is a full-stack Zerodha clone built using modern web technologies. The frontend is developed with React.js to create a responsive and interactive user interface. 
The backend is powered by Node.js and Express.js, providing robust APIs and server-side functionality.
MongoDB is used as the database to efficiently manage user information and trading data. The application replicates essential features of a stock trading platform with a scalable and well-structured architecture.
postman is used for API testing.
